# Backend: ASP.NET Core 9.0, Entity Framework Core, ASP.NET Core Identity.
# Data Storages: PostgreSQL, Redis, ElasticSearch.
# DevOps: Docker Compose, Nginx.
# Logging: Seq, Serilog.
# Frontend: Angular 18, SCSS.


## Start
1. Clone the repository.
2. Run `docker-compose up --build`.
3. Open `http://localhost:4200`.
